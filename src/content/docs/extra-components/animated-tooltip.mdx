## Usage Guide

Install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge framer-motion class-variance-authority
```

Next, add [**cn**](https://github.com/yogendrarana/craftui/blob/master/src/lib/utils.ts) utility:
```jsx
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```

## Props

### Tooltip

| Prop             | Type              | Description                                                         | Default    |
| ---------------  | ----------------- | ------------------------------------------------------------------- | ---------- |
| children         | ReactNode         | The content to be rendered inside this component                    | _          |
| delay?           | number            | Delay in milliseconds before the tooltip appears or disappears.     | 100        |
| className?       | string            | Custom class to apply to the tooltip.                               | _          |
| onClick?         | () => void        | Callback function to be called when the tooltip is clicked.         | _          |
