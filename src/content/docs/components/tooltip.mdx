## Usage Guide

Install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge
```

Next, add [**cn**](https://github.com/yogendrarana/craftui/blob/master/src/lib/utils.ts) utility:
```jsx
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```

## Props

### Tooltip

| Prop             | Type              | Description                                                         | Default    |
| ---------------  | ----------------- | ------------------------------------------------------------------- | ---------- |
| children         | ReactNode         | The element that triggers the tooltip when hovered or focused       | _          |
| variant?         | string            | "default" | "primary" | "secondary" | "destructive"                 | default    |
| direction?       | string            | "top" | "right" | "bottom" | "left"                                 | top        |
| delay?           | number            | Delay in milliseconds before the tooltip appears or disappears.     | 200        |

### TooltipTrigger

| Prop         | Type              | Description                                                               | Default  |
| -----------  | ----------------- | ------------------------------------------------------------------------- | -------- |
| children     | ReactNode         | The element that acts as the trigger for the tooltip.                     | _        |
| asChild?     | boolean           | If true, renders the children element as the trigger without wrapping it in an additional DOM element.               | _       |

### TooltipContent

| Prop           | Type              | Description                                                           | Default |
| -------------  | ----------------- | --------------------------------------------------------------------- | ------- |
| children       | ReactNode         | The content displayed inside the tooltip.                             | _       |
