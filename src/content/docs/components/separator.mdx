## Props

| Prop         | Type              | Description                                             | Default         |
| -----------  | ----------------- | ------------------------------------------------------- | --------------- |
| orientation  | string            | The content to be rendered inside the Tablist           | "horizontal"    |
| decorative   | boolean           | If true, the separator is decorative and does not contribute to accessibility features    | true            |
| className?   | string            | Optional class names to style the Separator component   | _               |
