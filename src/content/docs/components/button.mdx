## Usage guide
Install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge class-variance-authority
```

Next, add [**cn**](https://github.com/yogendrarana/craftui/blob/master/src/lib/utils.ts) utility:
```jsx
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```

## Props

| Prop             | Type              | Description                                                         | Default |
| ---------------  | ----------------- | ------------------------------------------------------------------- | ------- |
| children         | ReactNode         | The content to be rendered inside it                                | _       |
| variants         | string            | default, danger, outline, secondary, link                           | default |
| size             | string            | default, sm, lg, or icon                                            | default |
| asChild?         | string            | If false, returns button other wise returns children                | false   |
| className?       | string            | Optional class names to style the button component                  | _       |
