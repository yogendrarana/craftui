## Props

| Prop             | Type   | Description                                          | Default |
| ---------------- | ------ | ---------------------------------------------------- | ------- |
| pendulumCount    | number | number of pendulum bobs to be rendered               | 8       |
| swingDuration    | number | duration (in seconds) to complete one full swing     | 2       |
| swingAngle       | number | angle (in degrees) at which the pendulum bob swings. | 30      |
| threadLength     | number | length (in pixels) of the thread                     | 80      |
