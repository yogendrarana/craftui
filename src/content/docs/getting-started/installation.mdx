### Utility 
If a component relies on the **cn** utility, install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge
```

Next, add this utility function in your project at **/src/utils/cn.ts**:
```jsx filename="src/utils/cn.ts"
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```
