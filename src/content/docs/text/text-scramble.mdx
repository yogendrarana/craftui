## Usage Guide

Install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge
```

Next, add [**cn**](https://github.com/yogendrarana/craftui/blob/master/src/lib/utils.ts) utility function in your project:
```jsx
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```

## Props

| Prop             | Type      | Description                                | Default            |
|------------------|-----------|--------------------------------------------|--------------------|
| defaultText      | string    | The text to display initially.             | "Hello, World!"    |
| scrambleOnHover  | boolean   | Enables scrambling animation on hover.     | true               |
| duration         | number    | Duration of the scrambling animation (ms). | 1000               |
| characters       | string    | Custom characters used for scrambling.     | ""                 |
| className        | string    | Additional class names for styling.        | ""                 |
