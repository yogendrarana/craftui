## Usage Guide

Install the **clsx** and **tailwind-merge** packages:
```bash
npm install clsx tw-merge
```

Next, add [**cn**](https://github.com/yogendrarana/craftui/blob/master/src/lib/utils.ts) utility:
```jsx
import { twMerge } from "tailwind-merge";
import { clsx, type ClassValue } from "clsx";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```

## Props

### Drawer

| Prop           | Type                       | Description                                                                | Default         |
| -------------  | -------------------------- | -------------------------------------------------------------------------- | --------------- |
| children       | ReactNode                  | The content to be rendered inside the component                            | _               |
| origin?        | string                     | left, right, top, bottom                                                   | right           |
| variants?      | Object                     | Framer motion variants.                                                    | _               |
| transition?    | Object                     | Framer motion transition.                                                  | _               |
| defaultOpen?   | boolean                    | If true, the drawer is open by default.                                    | false           |
| open?          | boolean                    | Controls whether the dawer is open or closed.                              | undefined       |
| onOpenChange?  | (isOpen: boolean) => void  | Callback function triggered whenever the open state of the drawer changes. | undefined       |

### DrawerTrigger

| Prop         | Type              | Description                                             | Default |
| -----------  | ----------------- | ------------------------------------------------------- | ------- |
| children     | ReactNode         | The content to be rendered inside the drawer trigger    | _       |
| className?   | string            | Additional class names to style the DrawerTrigger       | _       |
| asChild?     | boolean           | If false, returns button other wise returns children    | false   |
