## API Reference

### useConfetti

Returns a function to trigger confetti animations.

### ConfettiOptions

| Prop          | Type   | Description                                                                            | Default |
| ------------- | ------ | -------------------------------------------------------------------------------------- | ------- |
| particleCount | number | The number of confetti and sequin particles                                            | 50      |
| origin        | string | top \| bottom \| left \| right \| top-left \| top-right \| bottom-left \| bottom-right | center  |
| duration      | number | Duration in milliseconds before the canvas is removed                                  | 3000    |
