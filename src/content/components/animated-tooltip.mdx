## Props

#### Tooltip

| Prop       | Type       | Description                                                     | Default |
| ---------- | ---------- | --------------------------------------------------------------- | ------- |
| children   | ReactNode  | The content to be rendered inside this component                | \_      |
| delay?     | number     | Delay in milliseconds before the tooltip appears or disappears. | 100     |
| className? | string     | Custom class to apply to the tooltip.                           | \_      |
| onClick?   | () => void | Callback function to be called when the tooltip is clicked.     | \_      |
