### Custom Particle Count

```tsx
import { useConfetti } from "@/hooks/use-confetti";

function CustomParticleCount() {
  const triggerConfetti = useConfetti();

  return (
    <button onClick={() => triggerConfetti({ particleCount: 100 })}>
      More Confetti!
    </button>
  );
}
```

## API Reference

### useConfetti

A hook that returns a function to trigger confetti animations. The confetti is rendered on a canvas element that is automatically created and removed.

**Returns:** `(options?: ConfettiOptions) => void`

### ConfettiOptions

Configuration options for the confetti animation.

| Parameter     | Type                                                                                                               | Description                              | Default  |
| ------------- | ------------------------------------------------------------------------------------------------------------------ | ---------------------------------------- | -------- |
| particleCount | number                                                                                                             | Number of confetti particles to generate | 50       |
| origin        | "top" \| "bottom" \| "left" \| "right" \| "center" \| "top-left" \| "top-right" \| "bottom-left" \| "bottom-right" | Starting position for the confetti       | "center" |
| duration      | number                                                                                                             | Duration in milliseconds before cleanup  | 3000     |
