## Props

#### Timeline

| Prop        | Type      | Description                                                  | Default    |
| ----------- | --------- | ------------------------------------------------------------ | ---------- |
| children    | ReactNode | The content to be displayed inside the component             | \_         |
| variant?    | string    | Accepted values are **vertical** or **horizontal**           | "vertical" |
| className?  | string    | Additional CSS classes to style the Timeline component.      | ""         |
| aria-label? | string    | Accessible label describing the timeline for screen readers. | ""         |
